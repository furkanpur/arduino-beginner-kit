# Using LCD Display

## Requirements

| Equipment | Quantity |
|---|:-:|
| 16x2 LCD Display + I2C Converter  | 1 |
| Bread Board  | 1  |
| Arduino Uno  | 1  |
| Wires  |   |

## Set Up
1. I2C Converter Module Connection
    
    | Pin Name | Pin Number |
    |---|:-:|
    | VCC      |    3.3 V |
    | Ground   |    Ground|
    | SDA      |    SDA|
    | SCL      |    SCL|
    
    P.S. : 16x2 LCD Display module has a I2C converter. 

P.S. : The latest version of the project is in media folder as a Fritzing Project.
