# Parking  Sensor 

## Requirements

| Equipment | Quantity |
|---|:-:|
| HC-SR04 ultrasonic sensor | 1 |
| Arduino Uno R3 | 1 |
| Buzzer | 1 |
| Wires | |
| Bread Board | 1 |
| LED | 2 |


### Set Up

1. HC-SR04 Sensor Connection
 
    | Pin Name | Pin Number |
    |---|:-:|
    | VCC | 5V |
    | Ground | GND |
    | Echo | Pin 12|
    | Trig | Pin 13|

2. LED Connection
 
    | Pin Name | Pin Number |
    |--- |:-:|
    | LED 1 | Pin 7 |
    | LED 1 | Pin 10 |

3. Buzzer Connection
 
    | Pin Name | Pin Number |
    |---|:-:|
    | Buzzer| Pin 6 |

P.S. : The latest version of the project is in media folder as a Fritzing Project.
