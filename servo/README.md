# Controlling Multiple Servo Motor By Using Joystick

## Requirements

| Equipment | Quantity |
|---|:-:|
| Micro Servo SG90 | 2 |
| Arduino MEGA | 1 |
| Joystick | 1 |
| Wires | |

## Set Up

1. First Micro Servo Motor SG90 Connection
 
    | Pin Name | Pin Number |
    |---|:-:|
    | Red Wire | 5 V |
    | Orange Wire | Pin 7|
    | Brown Wire | GND|

2. Second Micro Servo Motor SG90 Connection
 
    | Pin Name | Pin Number |
    |--- |:-:|
    | Red Wire | 5 V |
    | Orange Wire | Pin 8 |
    | Brown Wire | GND |

3. Joystick Connection
 
    | Pin Name | Pin Number |
    |---|:-:|
    | +5V| 5 V |
    | VRx | A0|
    | VRy | A1|
    | Ground| GND|


P.S. : The latest version of the project is in media folder as a Fritzing Project.
